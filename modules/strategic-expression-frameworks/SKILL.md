# 战略表达框架

本技能提供三种结构化表达框架，帮助应对不同挑战的沟通场景。

## 框架选择决策树

根据沟通挑战类型选择框架：

```
沟通挑战是什么？
├─ 需要切换视角（扩大/缩小视野）
│  ├─ 处理敏感/保密信息 → 变焦镜（拉远）
│  ├─ 证明决策合理性 → 变焦镜（拉远）
│  ├─ 反驳一概而论 → 变焦镜（拉近）
│  └─ 全局观妨碍看关键问题 → 变焦镜（拉近）
│
├─ 存在对立观点或极端选项
│  ├─ 需要迅速无可辩驳地回答 → 钟摆辩证（激进形式）
│  └─ 需要调和观点走向共识 → 钟摆辩证（温和形式）
│
└─ 需要让抽象概念变得生动易懂
   ├─ 解释技术/专业概念给外行 → 图景辅助（类比）
   └─ 用具体事实增强说服力 → 图景辅助（实例）
```

## 框架一：变焦镜表达

**适用场景**：复杂问题解释、敏感信息处理、决策合理性证明、反驳一概而论

**核心原理**：通过视野的动态切换（拉远/拉近），引导听众理解问题的全局意义或具体细节。

### 执行步骤

**A. 拉远镜头（扩大视野）**

1. 确定要说明的细节
2. 将细节与更宽广的大局联系起来
3. 说明细节在大局中的意义

应用场景：
- 说明细节与大局的关系
- 处理敏感/保密信息（先说明保密，再扩大到可谈论的宽泛问题）
- 证明选择/决定的合理性（先给具体回答，再在更广泛语境中说明意义）

**B. 拉近镜头（聚焦细节）**

1. 承认全局性观点或问题
2. 聚焦到具体、生动的事例或数据
3. 用细节推翻笼统假定

应用场景：
- 全局观妨碍看待关键问题时
- 反驳一概而论的说法
- 将宽泛认识具体化

**C. 动态切换**

透过同心圆（小→中→大，或大→中→小）带动听众，产生镜头移动的兴奋感和好奇心。

> 详细示例参见 [references/zoom-lens-examples.md](references/zoom-lens-examples.md)

## 框架二：钟摆辩证表达

**适用场景**：争议性话题、僵局打破、妥协方案提出、现状辩护

**核心原理**：在两个极端观点间摆动，最终引导听众接受合理的中间立场。

### 执行步骤

**A. 激进形式（推翻两端，捍卫中间）**

适用：面对棘手问题，需要迅速、无可辩驳的回答

1. 描述极端选项A及其负面后果
2. 描述极端选项B及其负面后果
3. 提出中间立场并证明其明智

**B. 温和形式（调和两端，提出妥协）**

适用：承认不同观点，走向中间立场或双赢

1. 承认观点A的合理性（"一方面..."）
2. 承认观点B的合理性（"另一方面..."）
3. 综合得出中间立场/双赢结果

### 话术过渡词

使用以下过渡词引导听众在两极间摆动：
- "从一方面看..."
- "反过来看..."
- "平心而论..."
- "一方面...另一方面..."

> 详细示例参见 [references/pendulum-dialectic-examples.md](references/pendulum-dialectic-examples.md)

## 框架三：图景辅助表达

**适用场景**：技术性话题解释、复杂概念传达、增强演讲生动性、非专业听众沟通

**核心原理**：通过类比或实例，将抽象概念转化为听众头脑中生动、难忘的视觉图景。

### 执行步骤

**A. 类比方式**

1. 识别听众完全理解的熟悉事物（源域）
2. 识别要解释的新思想/技术概念（目标域）
3. 建立两者之间的相似点

关键要求：类比物必须是听众完全理解的事物。

**B. 实例方式**

1. 提取核心观点
2. 提供清晰、具体、包含确切数据或生动细节的真实事例
3. 通过实例激发兴趣，帮助听众"体验"思想

关键要求：避免抽象的长篇解释，用具体数据和生动细节。

> 详细示例参见 [references/picture-assist-examples.md](references/picture-assist-examples.md)

## 组合使用建议

三种框架可组合使用以增强表达效果：

1. **先用图景辅助**建立听众理解基础
2. **再用变焦镜**在宏观意义和微观细节间切换
3. **遇到对立观点时用钟摆辩证**化解冲突

## 注意事项

- 变焦镜：确保视野切换的逻辑连贯性，不适用于简单的线性叙述
- 钟摆辩证：不适用于只有一方绝对正确的原则性/底线问题
- 图景辅助：不适用于纯数据或高度抽象的数学推导

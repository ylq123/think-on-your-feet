# 演讲结构与逻辑设计

## 何时使用
- 需要构建演讲/汇报的逻辑结构
- 需要向他人清晰传达思想或讲解复杂信息
- 需要构建完整、有说服力且令人难忘的呈现结构
- 面临即兴发言需要迅速组织思想

## 不适用场景
- 非正式的随意交谈或极短时间的电梯演讲
- 无需逻辑结构的简单寒暄

## 核心原则：清晰沟通三原则

### 1. 分析 - 直达核心
运用三部分分析，迅速切中议题核心。将话题组织成逻辑上相互衔接的片段。

### 2. 分离 - 将思想区隔分明
确保听众消化第一个思想后，再给第二个思想，避免信息混淆和超载。

### 3. 推进 - 让逻辑流畅
按逻辑顺序整理思想，使每个思想建立在上个思想基础上，形成向前推进的动力。

## 标准演讲格式

### 整体结构
```
介绍（5%时间）→ 主体三部分 → 结论（5%时间）
```

### 介绍部分
1. **界定话题：** 第一句话直接陈述核心主题标题
2. **宣布计划：** 给出三个主要部分的提纲
3. **暗示结论（可选）：** 若需制造悬念可省略

### 主体部分
- 第一部分：左脑"柱子"关键词 → 右脑展开细节
- 第二部分：左脑"柱子"关键词 → 右脑展开细节
- 第三部分：左脑"柱子"关键词 → 右脑展开细节

### 结论部分
1. **重述三部分计划：** 快速回顾关键词
2. **呼应标题：** 几乎一字不差地用核心标题结束

## "书立"法则

开头与结尾是记忆的锚点，需特殊处理：

| 要素 | 要求 |
|------|------|
| 长度 | 各不超过总时长的5% |
| 功能 | 凸显核心主题，让计划平稳推进 |
| 风格 | 展现艺术家风度，简练是机智的灵魂 |

**开头禁忌：** 切勿讲冗长拖沓的故事或牵强的笑话。好机长告诉目的地，不讲漏油故事。

## 逻辑计划选择

### 三大法则
1. **形式服从功能：** 确保计划强化预定结论
2. **变换计划类型：** 主计划与子片段计划应不同
3. **变换片段数量：** 让各片段数量在2-5个之间变动

### 双重测试选择最佳计划

**电影测试：** 计划是否简单、生动、富有推进力？

**结论测试：** 计划能否产生"对当前听众来说最强有力的结论"？

### 常用逻辑计划
- 三角计划：三个并列维度
- 时钟计划：时间顺序
- 钟摆计划：对比论证
- 地点计划：空间维度
- 变焦镜计划：从宏观到微观
- 收益计划：利益点罗列

### 复合技术
将两种或更多逻辑模式交叠运用，产生多重说服效果：
- 环球计划 + 变焦镜计划（全球→国家→城市）
- 时钟计划 + 正反计划（产生"季节性"计划）

## 克服"白纸瘫痪"

1. 将问题分拆为可管理的小块
2. 针对重要话题，运用3-4个计划加以评议
3. 画出图标，标出三个要点"柱子"
4. 迅速制定若干不同计划探索新角度
5. 从中选取最有力的一个

## 输出要求
- 经过电影测试和结论测试验证的逻辑结构
- 包含主计划与复合子计划
- 配备生动的"词语桩子"
- 首尾呼应的"书立"式开场结尾

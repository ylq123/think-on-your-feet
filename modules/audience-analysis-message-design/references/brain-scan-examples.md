# 大脑扫描图与标题编写示例

## 大脑扫描图案例

### 案例：向管理层汇报预算削减方案

| 维度 | 分析内容 |
|------|----------|
| 背景知识 | 管理层关注成本控制、了解行业下行趋势、重视团队稳定 |
| 希望 | 保持核心竞争力、避免人才流失、维持项目进度 |
| 恐惧 | 关键员工离职、项目延期、士气崩溃、被竞争对手超越 |

**核心主题提炼：** "精准瘦身，强健体魄"

### 案例：向技术团队推行新工具

| 维度 | 分析内容 |
|------|----------|
| 背景知识 | 熟悉现有技术栈、对变化有抵触、重视效率 |
| 希望 | 减少重复工作、提升代码质量、获得新技能 |
| 恐惧 | 学习成本高、短期效率下降、被替代 |

**核心主题提炼：** "三天上手，效率翻倍"

## 标题编写对比示例

| 差的标题 | 好的标题 | 改进原因 |
|----------|----------|----------|
| 关于优化报酬待遇方案的若干思考 | 加薪方案 | 具体、简短 |
| 在当前这个时候我们应该采取行动 | 现在行动 | 去除曲折陈述 |
| 据认为需要加强团队建设 | 我们要建强团队 | 主动语态 |
| 一位举止温文尔雅的人士来到了现场 | 绅士到场 | 名词动词主导 |
| 远途征程中我们需要做好准备 | 旅行前准备 | 短词替代 |

## 标题长度检验

好的标题应在3-7个词之间：
- ✓ "精准瘦身，强健体魄"（4词）
- ✓ "三天上手，效率翻倍"（4词）
- ✓ "质量优先，成本可控"（4词）
- ✗ "关于如何在当前经济形势下优化公司运营成本结构的战略思考"（过长）

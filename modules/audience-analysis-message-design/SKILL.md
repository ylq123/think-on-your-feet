# 听众分析与核心信息设计

## 何时使用
- 需要向特定听众传达关键要点或进行说服性沟通
- 准备商务演讲、工作汇报、营销沟通前
- 需要确保听众准确理解并接受观点

## 不适用场景
- 纯数据罗列或无需说服的例行通知
- 纯粹的信息广播或无需听众反馈的单向通知

## 执行流程

### 第一步：转变沟通心智
将自己定位为"帮助听众接收思想的引导者"，而非"传达者"。避免"爱因斯坦情结"和自我陶醉，接收意味着理解。

### 第二步：实施"大脑扫描图"（限时3分钟）
在纸上画出包含三个维度的表格：

| 维度 | 记录内容 |
|------|----------|
| 背景知识 | 听众对话题的共同假定、价值判断或经历 |
| 希望 | 听众绝对诚实时的积极情感与期待 |
| 恐惧 | 听众绝对诚实时的担忧与负面反应 |

**操作方法：**
1. 3分钟内在每个标题下快速写下脑中浮现的信息
2. 圈出显眼的关键词
3. 写下听众可能会提出的问题
4. 找出潜在核心主题
5. 着重研究希望与恐惧，了解听众兴趣并突出相似之处

**信息来源：**
- 根据指示：询问客户/上司听众最看重什么
- 根据经验和常识：本能选定目标准确的核心主题
- 对听众进行"大脑扫描"

### 第三步：提炼关键要点
关键要点 = 对当前听众最重要的事项。必须用听众的耳朵和头脑思考，告诉听众想听的东西，找到切入话题的正确角度。

### 第四步：编写核心标题（理想长度：3-7个词）

**词语选择法则：**
- 选用熟悉词而非生僻词（说"死亡"，不说"过世"）
- 选用具体词而非抽象词（说"加薪"，不说"报酬待遇方案"）
- 选用单词而非曲折陈述（说"现在"，不说"在当前这个时候"）
- 选用短词而非长词（说"旅行"，不说"远途征程"）

**词性法则：** 去除形容词与副词，充分利用名词和动词
- 用"一位绅士"代替"一位举止温文尔雅的人士"
- 用"他向前冲"代替"他飞快地奔跑"

**语态法则：** 使用主动语态增加力量和热情
- 说"我认为"，不说"据认为"
- 说"他击中球"，不说"球被他所击中"

### 第五步：实施有说服力沟通三步法
1. **集中（主题）：** 找到适合听众的主题，问"我的听众想知道些什么？"
2. **澄清（计划）：** 构建令人信服的逻辑计划来讲授主题
3. **影响（反应）：** 监控听众反应，据此调整表达

## 输出要求
- 精准的听众画像（背景知识、希望、恐惧）
- 符合"熟悉、具体、简短、名词动词主导、主动语态"原则的3-7词核心标题
- 切中听众痛点与诉求的沟通策略
